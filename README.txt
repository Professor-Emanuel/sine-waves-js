A lightweight graphical user interface for changing variables in JavaScript.
Can be installed or added as a script from:
dat.gui: https://www.npmjs.com/package/dat.gui

If you install it, add in your JS file at the top: import * as dat from 'dat.gui'

I used:
<script src="https://cdnjs.cloudflare.com/ajax/libs/three.js/r83/three.js"></script>
<script src="https://cdnjs.cloudflare.com/ajax/libs/dat-gui/0.7.2/dat.gui.js"></script>
in sine_waves.html head tag, since:
<script type="text/javascript" src="dat.gui.min.js"></script>
generated errors
